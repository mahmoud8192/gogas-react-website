# GoGas Company Website (Frontend)

- A responsive company website built for GoGas.
  This project focuses on the frontend implementation using React and Tailwind CSS.

 > Note: This project is a frontend implementation for portfolio purposes .

 ## Tech Stack
- React
- Tailwind CSS
- Vite
- React Router

## Features
- Fully responsive design
- Mobile-first layout
- Reusable React components
- Clean UI with Tailwind utility classes
- Accessible navigation

  ## What I Learned
- Structuring a real company website in React
- Creating consistent design systems with Tailwind
- Handling responsiveness across devices
